Getting started

Choose a demo home page to getting started, upload template file to web hosting.

Please do not edit style in theme-*.css, alway place customize style at assets/css/custom.css.
template use customize bootstrap, do not replace with cdn.
*.html - template html /assets/ /php/ contact.php - contact form mailchimp.php - mailchimp subscribe form /img/ /* template image */

Theme style

Open *.html find below code and edit the css name

<!-- edit to theme-1.css theme-2.css theme-3.css theme-4.css theme-5.css theme-6.css theme-7.css --> <link id="theme" rel="stylesheet" href="assets/css/theme-1.css">
If we plan to use same style at all page, the faster method is just change the css file name to theme-1.css, so we do not need edit above code.

Button

Template use btn-theme-1 and btn-theme-2 to control button color, for example

HTML markup <a href="#" class="btn btn-lg btn-pill btn-shadow btn-theme-1"><i class="icon icon-left fa fa-fw fa-arrow-right"></i> Button</a>
.btn-lg .btn-pill .btn-shadow just an options.
You may still use bootstrap button class to control the button style.

Navbar - Style

Add below class to #siteHeaderNavbar to control the navbar style.

.navbar-bg-from-transparent - when the page not yet scroll, the navbar will be transparent background. .navbar-fg-from-light - when the page not yet scroll, the navbar will light text, use with .navbar-bg-from-transparent .navbar-fg-from-dark - when the page not yet scroll, the navbar will dark text, use with .navbar-bg-from-transparent If no above class added, navbar will use default from below class. .navbar-default - default style .navbar-default - dark style

Module - Background overlay

HTML markup inside .module or #pageIntro <div data-css-opacity=".4" class="bg-overlay bg-black"></div>
Use Color Utility Class to control the background color.
Use data-css-opacity to control the opacity.

Module - Background image

HTML markup inside .module or #pageIntro <div data-css-background-image="assets/img/bg-img.jpg" class="bg-img bg-img-parallax"></div>
Use data-css-background-image to control the image src.
Use .bg-img-parallax to enable the parallax effect on desktop device.

Module - Background slideshow

HTML markup inside .module or #pageIntro <div data-bg-slideshow-src="assets/img/bg-slideshow-%.jpg" data-bg-slideshow-amount="3" data-slideshow-delay="7000" data-bg-slideshow-animation="kenburns" class="bg-slideshow"></div> Setting data-bg-slideshow-src - image src, do not replace % data-bg-slideshow-amount - image amouth data-bg-slideshow-animation - enable kenburns animation data-slideshow-delay - delay time Above code we have set 3 image for background slideshow assets/img/bg-slideshow-1.jpg assets/img/bg-slideshow-2.jpg assets/img/bg-slideshow-3.jpg
Use .bg-slideshow-no-timer to disable the slideshow timer.

Module - Background youtube

HTML markup inside .module or #pageIntro <div class="bg-youtube"> <div data-css-background-image-mobile="assets/img/page-intro-bg-youtube-fallback.jpg" class="bg-youtube-fallback"></div> <div data-css-background-image-desktop="assets/img/page-intro-bg-youtube-placeholder.jpg" class="bg-youtube-placeholder"></div> <div data-property="{videoURL:'https://www.youtube.com/watch?v=eeUqk7TGFtc',startAt: 136,stopAt: 156,autoPlay: true,mute: true,showControls: true, quality: 'highres', stopMovieOnBlur: false, playlistURL: null, containment:'self', showYTLogo: false }" class="bg-youtube-player"></div> </div> Setting { videoURL: 'https://www.youtube.com/watch?v=eeUqk7TGFtc', // Video url startAt: 136, // Start time, 0 to start stopAt: 156, // End time, 0 to ignore autoPlay: true, // auto play mute: true, // muted at start showControls: true, // Show control } Setting data-css-background-image-mobile - fallback image src data-css-background-image-desktop - placeholder image src

Popup - Youtube video

HTML markup <a href="#" data-mfp-src="https://www.youtube.com/watch?v=LViVa-kKqzA" class="popup-youtube">Popup Youtube Video</a> Setting data-mfp-src - youtube video url

FAQs

How to disable #pageIntro overlay like demo?
Please check Overlay Section, if you need disable overlay at all page, you may just add #pageIntro .bg-overlay { display: none !important; } to assets/css/custom.css, so no need to edit the HTML markup.
